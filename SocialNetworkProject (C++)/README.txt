The project is a mock social network with a Model-View-ViewModel (MVVM) interface. It uses C++11 libraries and features such as smart pointers and regex.
The project uses the beta library for filesystem of C++20 implemented by Microsoft, it may not work correctly with the newer versions.
The code is English, however, the text interface is in Italian due requirements of the course
